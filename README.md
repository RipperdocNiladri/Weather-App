# Weather App

A simple Python Weather App using the OpenWeatherMap API.

## Features

- User Time & Date
- Location Time & Date
- City, Country
- Current Weather
- Coordinates(Latitude & Longitude)
- Temperature
- Humidity
- Wind Speed
- Wind Direction
- Visibility
- Pressure
- Sunrise
- Sunset

## Installation

pip install -r requirements.txt

Create a .env file

OPENWEATHER_API_KEY=your_api_key

Run

python weather.py
